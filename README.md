# QuickStack - Akıllı Pano ve Dosya Yönetimi

QuickStack, kopyaladığınız her şeyi (metin, kod, bağlantı, görsel) otomatik olarak kaydeden ve düzenleyen akıllı bir pano yöneticisidir.

## Özellikler

*   **Arka Planda Çalışma (Sistem Tepsisi):** Uygulamayı kapattığınızda tamamen kapanmaz, sistem tepsisinde (tray) çalışmaya devam eder. Tamamen çıkmak için tepsi ikonuna sağ tıklayıp "Çıkış" diyebilirsiniz.
*   **Otomatik Başlangıç (Autostart):** Windows açılışında otomatik olarak başlar. Açılışta ekranı rahatsız etmemek için arka planda (tepside) gizli olarak başlar.
*   **Kolay Erişim Kısayolları:** Aşağıdaki tuş kombinasyonlarıyla uygulamayı anında açabilirsiniz:
    *   `Windows Tuşu + Shift + V` (Önerilen)
    *   `Windows Tuşu + Alt + V`
    *   `Windows Tuşu + Shift + C`
    *   `Ctrl + Shift + C`

## Geliştirme

Projeyi yerel olarak çalıştırmak için:

```bash
npm run tauri dev
```
